Badge cosmetics

- Recommended size: 256x256 transparent PNG.
- Current placeholder files can be replaced directly with final badge art.
- Use the same filename as the badge id, for example `first_win.png` or `br_winner.png`.
- New badge ids should be added to `BADGE_CATALOG` in `app.js`.
- Dynamic win badges such as `br_wins_live.png` and `dm_wins_live.png` use the PNG as the base art, then the website overlays the current win number.
