Getting Started

How to play

  1. Join the Discord to get the current server address, announcements, match times, rules, and updates.
  2. Install the Call of Block modpack from CurseForge or Modrinth so the launcher keeps every dependency matched.
  3. Open the server, queue for Battle Royale or Deathmatch, and pick the team or kit shown in-game.
  4. For Battle Royale, drop from the fly-by, loot Shmar, watch the zone, and survive to the final fight.
  5. For Deathmatch, choose a kit, play with your team, and reach the target score before the other side.

Questions

FAQ

Is this vanilla Minecraft?

No. Call of Block is a modded server and needs the matching modpack version.

What version is the server on?

The server runs Minecraft 1.20.1 on Forge.

Can cracked players join?

The server can allow cracked players, but the modpack is still required.

What modes are available?

Battle Royale and Deathmatch are the main modes right now. Vehicle-focused modes are planned later.

Where do stats come from?

Stats are saved by the server after matches and shown here for leaderboards, profiles, weapons, maps, and match history.

Server Help

Commands and controls

Type /brhelp in-game for the short version. Admins can open the full setup reference on its own page.

Controls
  • Open Call of Block MenuBind this in Minecraft Controls, or use /brmenu.
  • /brmenuOpens the main server menu for queues, teams, votes, kits, and mode panels.
  • /hubReturns you to hub. If you are spectating Battle Royale, it exits spectator first.
Battle Royale
  • /brmenuJoin or leave the BR queue, pick solo/team, create a team, and check match state.
  • /brspectateJoins a running BR match as a spectator when you are not a participant.
  • /brspectate leaveLeaves BR spectator mode and returns you to normal hub flow.
Deathmatch
  • /joindmJoins the Deathmatch queue.
  • /leavedmLeaves the Deathmatch queue and sends you back to hub.
  • /dmteam 1|2|randomSets your preferred DM team before teams are locked.
  • /vote map <id>Votes for a map while queued. Use /vote random to vote random.
  • /kitOpens the kit picker during an active DM round. Use /kit <id> to select directly.
Requirements

Player commands require being on the server. Voting requires being queued for Deathmatch. Kit changing requires an active Deathmatch round. Admin commands require permission level 2.

Admin Reference

Setup and control commands

These commands require permission level 2 unless noted. Use /bradmin doctor after edits to check the important server pieces.

Player help

Core Checks

  • /bradmin doctorChecks hub, arena, reset region, datapack, TACZ loot pool integrity, DM maps, lobbies, and Discord sync.
  • /bradmin sethubSaves the current player position as the hub spawn.
  • /bradmin mode hub|battle_royale|deathmatchManually sets the server mode state.
  • /bradmin reloadkitsReloads kit configuration.
  • /kit save <id> [displayName]Saves your current inventory as a kit.
  • /kit overwrite <id> [displayName]Overwrites an existing kit with your current inventory.
  • /kit delete <id>Deletes a kit.
  • /kit give <id>Applies a kit to yourself immediately for admin debugging, even outside an active DM round.

Battle Royale Runtime

  • /bradmin br startStarts Battle Royale from the current queued players and setup config.
  • /bradmin br endForces the running BR match into end/celebration cleanup.
  • /bradmin br cleanupofflineRemoves offline BR participants from runtime state.
  • /bradmin br cleandisconnectedAlias for cleanupoffline.
  • /bradmin bridge br_start|br_endLegacy/datapack bridge aliases for BR start/end.

BR Config

  • /bradmin br config showShows loot pool, chest range, vehicle range, zone, respawn, prep, and bus settings.
  • /bradmin br config reloadReloads BR setup config.
  • /bradmin br config chests <min> <max>Sets chest placement count range.
  • /bradmin br config vehicles <type> <min> <max>Sets vehicle spawn range for a vehicle type.
  • /bradmin br config preset <name>Applies a BR round preset.
  • /bradmin br config pool <id>Selects the main BR loot pool.
  • /bradmin br config zonepreset <id>Selects the zone preset.
  • /bradmin br config usezone on|offToggles BR zone usage.
  • /bradmin br config respawn on|offToggles BR redeploy respawns.
  • /bradmin br config prepchunks <value>Sets prep/reset chunk work per step.
  • /bradmin br config prepticks <value>Sets ticks between prep/reset steps.
  • /bradmin br config bus dropallowed get|set <value>Reads or sets the allowed bus drop distance.
  • /bradmin br config bus centeralign get|set <value>Reads or sets max bus center alignment.
  • /bradmin brconfig ...Legacy alias for most BR config commands, including wallheight/zonewallheight.

Arena And Zone

  • /bradmin zone setarena <radius> <margin>Saves BR arena center at your current position.
  • /bradmin zone reloadReloads zone phase config.
  • /bradmin zone configstartBegins the configured zone run from your position.
  • /bradmin zone begin <seconds> <targetRadius> <damage>Starts a manual random zone shift.
  • /bradmin zone stopStops the active zone.
  • /bradmin brrulesdisplay statusShows BR rules display status.
  • /bradmin brrulesdisplay setherePlaces the BR rules display at your position.
  • /bradmin brrulesdisplay refreshRefreshes the BR rules display.
  • /bradmin brrulesdisplay clearClears the BR rules display.

Reset Region

  • /bradmin resetregion set1Sets reset region corner one at your position.
  • /bradmin resetregion set2Sets reset region corner two at your position.
  • /bradmin resetregion showShows the selected reset region.
  • /bradmin resetregion save [chunksPerStep] [ticksBetweenSteps]Saves a reset snapshot for the selected region.
  • /bradmin resetregion start [chunksPerStep] [ticksBetweenSteps]Restores the saved snapshot.
  • /bradmin resetregion cancelCancels reset work.
  • /bradmin resetregion statusShows reset worker status.

Deathmatch Config

  • /bradmin dm config showShows DM time, kill limit, celebration, vote, and block-damage settings.
  • /bradmin dm config reloadReloads DM config.
  • /bradmin dm config time <seconds>Sets DM time limit.
  • /bradmin dm config kills <kills>Sets DM kill limit.
  • /bradmin dm config celebration <seconds>Sets end celebration length.
  • /bradmin dm config vote <seconds>Sets vote duration.
  • /bradmin dm config blockdamage on|offToggles DM block damage config.
  • /bradmin dmconfig ...Legacy alias for DM config commands.

Deathmatch Maps

  • /bradmin dmmap listLists DM maps.
  • /bradmin dmmap show <id>Shows one DM map.
  • /bradmin dmmap create <id> [displayName]Creates a DM map.
  • /bradmin dmmap delete <id>Deletes a DM map.
  • /bradmin dmmap setlobby <id> <lobby>Assigns a lobby to a map.
  • /bradmin dmmap clearlobby <id>Clears a map lobby override.
  • /bradmin dmmap addspawn <id> <team>Adds a team spawn at your position.
  • /bradmin dmmap removenearestspawn <id> <team>Removes the nearest spawn for a team.
  • /bradmin dmmap tprandom <id> <team>Teleports you to a random spawn for testing.

Deathmatch Lobbies

  • /bradmin dmlobby show <id>Shows a DM lobby.
  • /bradmin dmlobby create <id> [displayName]Creates a DM lobby.
  • /bradmin dmlobby delete <id>Deletes a DM lobby.
  • /bradmin dmlobby setspawn <id>Sets lobby spawn at your position.
  • /bradmin dmlobby tp <id>Teleports you to a lobby.
  • /bradmin dmlobby setgeneral <id>Sets the general DM lobby used by the queue.
  • /dmqueueShows the queued DM players.
  • /dmqueue sendmainSends queued DM players to the general lobby.

DM Runtime And Votes
